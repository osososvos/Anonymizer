Anonymizer

Es una aplicación Python diseñada para eliminar la Información Personal Identificable (PII) de documentos de Microsoft Word (.docx). La aplicación utiliza SpaCy para el Procesamiento del Lenguaje Natural (NLP) para identificar entidades PII y luego las redacta para garantizar la privacidad. La herramienta es fácil de usar, con una interfaz gráfica de usuario (GUI) construida con Tkinter.

Características

  • Cargar documentos de Word.
  • Identificar PII usando reconocimiento de entidades nombradas (NER) con SpaCy y expresiones regulares personalizadas.
  • Redactar PII en los documentos.
  • Guardar documentos redactados.
  • Modo Visual Mejorado: Activar un modo accesible para usuarios con discapacidad visual, ofreciendo mayor contraste y texto más grande.

Anonimizar
